## üéÆ GAME OVERVIEW

**Sequence Rush** is a hyper-casual mobile game that combines memory challenges with time-pressure reflex gameplay.

### Core Gameplay:
1. **Memorize:** Watch a sequence of colored buttons light up (3-5 seconds)
2. **Execute:** Reproduce the sequence before time runs out (10-15 seconds)
3. **Progress:** Complete 100+ levels with increasing difficulty

### Key Features:
- ‚ú® Unique blend of memory + reflex gameplay
- üéØ Easy to learn, hard to master
- ‚ö° Quick 15-30 second sessions
- üé® 8 beautiful unlockable themes
- üí∞ Fair free-to-play monetization
- üìà Daily challenges and achievements
- üèÜ Global leaderboards
